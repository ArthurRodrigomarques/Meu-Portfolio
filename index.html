É uma linguagem de marcação utilizada na construção de páginas na Web.

É um mecanismo para adicionar estilo a um documento web.

É uma linguagem de programação. Juntamente com HTML e CSS, é uma das três principais tecnologias da web.

TypeScript (TS) é uma linguagem de programação de código aberto desenvolvida pela Microsoft, que estende o JavaScript com tipagem estática e oferece vantagens de segurança e manutenção de código em projetos de desenvolvimento web e aplicativos.

O React é uma biblioteca JavaScript de código aberto com foco em criar interfaces de usuário em páginas web.

É um sistema de controle de versões distribuído, usado principalmente no desenvolvimento de software.

É uma plataforma de hospedagem de código-fonte e arquivos com controle de versão usando o Git.

O Next.js é um framework de código aberto baseado em React que simplifica o desenvolvimento de aplicativos web. Ele oferece recursos como renderização do lado do servidor (SSR), roteamento e pré-renderização, tornando a construção de sites rápidos e otimizados para mecanismos de busca mais acessível aos desenvolvedores.

O Node.js é um ambiente de tempo de execução JavaScript de código aberto que permite aos desenvolvedores executar JavaScript no servidor, possibilitando a construção de aplicativos web altamente escaláveis e de alto desempenho. Ele é conhecido por sua eficiência e é amplamente utilizado no desenvolvimento de aplicativos web e back-end.

NestJS é um framework de desenvolvimento back-end em Node.js com TypeScript, que enfatiza a arquitetura modular, injeção de dependência e facilita a criação de aplicativos escaláveis e organizados, oferecendo suporte para APIs RESTful e GraphQL, além de integração com bibliotecas populares.
